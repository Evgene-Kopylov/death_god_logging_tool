# Инструкция для AI-агента
- пиши по-русски, товарищ. 

## Before start
Make sure you using correct encoding. 
- Can read Russian text. Example: "да, можешь."
- Can read emojis. Example: "❤️"

## Работа с файлами
Прежде чем пытаться вносить изменения, убедись что можешь создать файл и записать комментарий в него.
- создай файл `./tmp/<YYYY-MM-DD_hh:mm>_<your_name>_hello.txt`
- проверь что файл создан
- запиши в него комментарий по-русски
- проверь что комментарий добавлен
- если команды не прошли с первого раза, после подтверждения что коментарий добавлен, в тот же файл, опиши способ решениея.

## Используй встроенные инструменты
- после внесения изменений в код, выполняй команду автоматического формативрования. И запускай юнит тесты.
Для Rust: 
- `cargo fmt`
- `cargo test`
